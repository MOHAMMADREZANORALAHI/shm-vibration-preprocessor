# SHM Vibration Preprocessor

Professional toolkit for vibration data cropping, plotting, and advanced preprocessing for Structural Health Monitoring (SHM).

## Features
- Signal Cropping Tool
- Interactive Vibration Plotter
- Advanced Dashboard with filtering
- FDD Modal Analysis
- Batch Feature Extraction

## Tools
- Python GUI applications
- Batch processing scripts
- Windows .bat launchers

## Installation
```bash
pip install -r requirements.txt
```

## Usage
Run the .bat files for easy launch.
